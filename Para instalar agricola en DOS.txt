#cd\progra~1\r\R-2.6.0dev\bin

RCMD check agricolae
RCMD INSTALL agricolae
RCMD Rd2dvi --pdf agricolae

Para hacer el manual en pdf
crear un nuevo folder.
md \localtexmf\tex

Paso 2 copiar de R-2.4.1\share\textmf

copy *.* \localtexmf\tex

Paso 3 ir a options de MikTex, Update y luego Refresh

Estos pasos se realizan cada vez que se instala un nuevo paquete.
